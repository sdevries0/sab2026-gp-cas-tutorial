Genetic Programming for Complex Adaptive Systems

A tutorial at the Simulating Adaptive Behaviour (SAB) conference 2026.

Overview

This tutorial introduces how to integrate the Kozax genetic programming framework with the ABMax framework from the JAX ecosystem for complex adaptive systems modelling.

Who should attend?

  • Researchers interested in adaptive behaviour and evolutionary computation
  • Practitioners building simulation workflows in JAX
  • Students learning model discovery with genetic programming

Topics

  • Fundamentals of genetic programming for symbolic model discovery
  • Integrating Kozax into ABMax pipelines
  • Hands-on setup and reproducible experimental workflows
